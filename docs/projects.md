# 🚀 项目演示

> 这里展示了博客集成的所有互动应用和 AI 系统

## 📋 项目列表

### 🚨 摔倒检测系统

**实时监控系统，基于 3D 点云技术识别摔倒行为**

- 🔍 **功能特点**：
  - 20 FPS 流畅的 3D 点云动画
  - 自动识别摔倒姿态并分级预警
  - 实时统计和数据分析
  
- 🛠️ **技术栈**：Streamlit + Plotly + Open3D
- 🌐 **访问地址**：
  - [Nginx 代理](/fall-detection/)（推荐）
  - [直接访问](http://localhost:8501)（需要单独启动服务）

---

### 🧠 RAG 知识库

**检索增强生成的智能问答系统**

- 🔍 **功能特点**：
  - 文档检索与智能问答
  - 支持多种文件格式
  - 基于向量相似度搜索
  
- 🛠️ **技术栈**：LangChain + ChromaDB + Streamlit
- 🌐 **访问地址**：
  - [Nginx 代理](/rag/)（推荐）
  - [直接访问](http://localhost:7860)（需要单独启动服务）

---

### 😊 情感分析系统

**文本情感识别与分析工具**

- 🔍 **功能特点**：
  - 实时情感极性分析（积极/消极/中性）
  - 情绪强度评分
  - 历史数据分析
  
- 🛠️ **技术栈**：FastAPI + Transformers + Vue.js
- 🌐 **访问地址**：
  - [Nginx 代理](/emotion/)（推荐）
  - [直接访问](http://localhost:8001)（需要单独启动服务）

---

### 🎮 AI 俄罗斯方块

**AI 智能体自主学习的经典游戏**

- 🔍 **功能特点**：
  - AI 自主学习游戏策略
  - 实时显示学习进度
  - 人类 vs AI 对战模式
  
- 🛠️ **技术栈**：Pygame + Reinforcement Learning + Flask
- 🌐 **访问地址**：
  - [Nginx 代理](/tetris/)（推荐）
  - [直接访问](http://localhost:5000)（需要单独启动服务）

## 🏗️ 架构说明

```
┌─────────────────────────────────────┐
│         Nginx (Port 80)             │
│    统一入口 + 反向代理 + HTTPS      │
└──────────────┬──────────────────────┘
               │
    ┌──────────┼──────────┬──────────┬────────────┐
    │          │          │          │            │
    ▼          ▼          ▼          ▼            ▼
┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐ ┌──────────┐
│ 博客   │ │ 摔倒   │ │  RAG   │ │ 情感   │ │   AI     │
│ :8080  │ │ 检测   │ │ 知识库 │ │ 分析   │ │ 俄罗斯   │
│        │ │ :8501  │ │ :7860  │ │ :8001  │ │ 方块     │
│        │ │        │ │        │ │        │ │ :5000    │
└────────┘ └────────┘ └────────┘ └────────┘ └──────────┘
```

所有服务通过 Docker Compose 编排，Nginx 提供统一的访问入口和负载均衡。

---

## 🚀 快速启动

```bash
cd /root/.openclaw/workspace/blog

# 一键启动所有服务
docker compose --profile with-rag --profile with-fall-detection up -d

# 查看服务状态
docker compose ps

# 查看特定服务日志
docker compose logs -f fall-detection
```
