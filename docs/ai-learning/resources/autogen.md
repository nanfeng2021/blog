## 框架概述

**AutoGen** 是微软推出的多 Agent 协作框架，支持通过自然语言对话完成复杂任务。

## 核心特性

### 多 Agent 协作

定义不同角色的 Agent，通过对话协同工作

### 代码执行

自动编写和执行 Python 代码解决问题

### 工具集成

无缝集成搜索引擎、API、数据库等外部工具

### 人机协作

人类可以作为 Agent 之一参与对话流程

## 应用场景

- 自动化数据分析
- 复杂问题求解
- 代码开发与调试
- 研究文献调研
