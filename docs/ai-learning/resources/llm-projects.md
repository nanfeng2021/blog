## 项目概述

通过实际项目开发，掌握 LLM 应用的核心技术和最佳实践。

## 入门项目

### 智能对话机器人

使用 OpenAI API 或开源模型构建聊天助手

### 文本摘要工具

实现新闻文章自动摘要功能

### 代码生成助手

基于 Copilot 思路的代码补全工具

## 进阶项目

### RAG 问答系统

结合向量数据库的知识库问答

### Agent 工作流

多步骤任务自动化处理

### 微调定制模型

在特定领域数据上微调开源模型
