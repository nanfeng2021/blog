## 教程概述

**scikit-learn** 是 Python 最流行的机器学习库，提供简洁统一的 API 接口。

## 核心内容

### 监督学习

分类（SVM、随机森林、逻辑回归）、回归（线性回归、岭回归）

### 无监督学习

聚类（K-Means、DBSCAN）、降维（PCA、t-SNE）

### 模型评估

交叉验证、网格搜索、性能指标

## 学习建议

1. 完成官方教程示例
2. 在 Kaggle 数据集上实践
3. 阅读源码理解实现
