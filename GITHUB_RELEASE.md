## 🌟 亮点功能

### 🔒 安全与 SEO
- ✅ HTTPS (A+ 评级)
- ✅ sitemap.xml + robots.txt
- ✅ Open Graph / Twitter Card
- ✅ 自定义 OG 预览图

### 🚀 核心功能
- ✅ **Giscus 评论系统** - 基于 GitHub Discussions
- ✅ **Umami 统计分析** - 隐私友好 + 事件追踪
- ✅ **站内搜索** - 全文检索（Ctrl+K）
- ✅ **阅读进度条** - 蓝橙渐变动画

### 🎨 视觉体验
- ✅ 品牌色系统（科技蓝 + 温暖橙）
- ✅ 移动端深度优化
- ✅ 暗黑模式适配
- ✅ Hero 区域 + 特性卡片

## 📦 部署方式

```bash
# 自动化部署
./deploy.sh

# 或手动部署
cd /root/.openclaw/workspace/blog
npm run build
sudo cp -r docs/.vitepress/dist/* /var/www/ainanfeng.cn/
```
